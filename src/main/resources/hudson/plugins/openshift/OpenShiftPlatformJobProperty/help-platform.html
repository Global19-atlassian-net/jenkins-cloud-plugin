Important: This setting only applies if you set the job Label Expression to match the Project name of your job.

Setting the job label is done automatically for you by the OpenShift client tools, but if you are manually setting up a job, you need to set this. You can set the Label Expression by checking Restrict where this project can be run and then copying your Project Name into the text field. The label is used to lookup the job details, including the Builder Type, Application UUID and Builder Size when your slave builders are created.

This property specifies the platform your application is running on.
